はじめての方へ

このページでは、初めてご利用される方のために、システムの使い方をご説明します。

1. 会員登録

  まずは会員登録を行ってください。会員登録には、メールアドレス(@前がidとして設定される)とパスワードが必要です。

  今、登録を行うには、以下のメールアドレスに連絡ください。

cistlicense@gmail.com

2. ログイン

  登録が完了したら、ログイン画面からid（メールアドレスの@前まで）とパスワードを使用してログインしてください。

3. 予算

  ログイン後、DBにアクセスして、各種ライセンス管理機能を利用できます。詳しい操作方法は以下をご参照ください。

4. サポート

  困ったことがあれば、メインメニューからお問い合わせいただけます。
